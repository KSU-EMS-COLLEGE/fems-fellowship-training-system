# منصة تدريب الزمالة - النسخة النهائية النظيفة

هذه النسخة مبنية على نسخة المنصة الاحتياطية التي أرسلتها، مع الاحتفاظ ببيانات الطلاب والروتيشنات الأساسية، وإضافة:

- صفحة إدارة رئيسية `index.html`
- صفحة تقييم خارجية `evaluation.html`
- روابط تقييم خاصة لكل طالب/روتيشن
- تحديث مباشر عبر Firebase Firestore
- أرشيف تقييمات
- تصدير CSV

## مهم
تم حذف أرقام الهوية والجوالات والإيميلات من ملف البيانات المرفوع للكود، حتى لا تظهر بيانات شخصية في مستودع GitHub العام.

## طريقة الرفع
ارفع جميع الملفات كما هي إلى GitHub أو Firebase Hosting.

## ربط Firebase
1. أنشئ مشروع Firebase.
2. فعّل Firestore Database.
3. من Project Settings > Web App انسخ firebaseConfig.
4. ضع القيم في ملف `app-config.js`.
5. انسخ محتوى `firestore.rules` إلى Firestore Rules ثم Publish.

## التشغيل
- افتح `index.html` من الموقع.
- اضغط أي خلية في جدول متابعة الإنجاز.
- أنشئ رابط تقييم داخلي أو خارجي.
- افتح الرابط من صفحة المقيم.
- بعد الإرسال تظهر الدرجة مباشرة في جدول الإنجاز.

## تنبيه أمني
هذه النسخة تعمل بدون تسجيل دخول وبدون رمز تحقق بناءً على طلبك. أي شخص يملك رابط الإدارة يمكنه رؤية بيانات المنصة، وأي شخص يملك رابط التقييم يمكنه تعبئته.
